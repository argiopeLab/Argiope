# Argiope
Argiope is a desktop suite for standardizing and analyzing images. AI-assisted tools are integrated in both modules to support image calibration, image segmentation and classification. Argiope supports RGB, multispectral, and hyperspectral imagery. The Standardize Images and Analyze Images modules are independent.

## How to report a bug

1. Open a new issue: https://github.com/argiopeLab/Argiope/issues/new
2. Describe:
   - The version of the software;
   - Your operating system;
   - The steps to reproduce the problem;
   - What you expected to happen;
   - What actually happened (error messages, screenshots, etc.).
